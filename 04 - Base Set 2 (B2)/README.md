# Base Set 2 (B2)

A compilation set released on February 24, 2000, combining cards from Base Set and Jungle into a single 130-card set. Base Set 2 made popular cards more accessible to players who missed the original releases.

## Set Information

| Property | Value |
|----------|-------|
| Set Code | base2 |
| Set Name | Base Set 2 |
| Series | Original |
| Release Date | February 24, 2000 |
| Total Cards | 130 |

## Notable Cards

- **Charizard** (#004) - The iconic card reprinted for a new generation
- **Blastoise** (#002) - Rain Dance remained a top-tier strategy
- **Wigglytuff** (#019) - Do the Wave continued to dominate
- **Scyther** (#017) - Free retreat Basic remained essential
- **Professor Oak** (#116) - Core draw power for all decks
