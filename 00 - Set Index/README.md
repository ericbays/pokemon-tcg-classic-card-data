## Card ID Format

All cards use a unique identifier: `{setCode}-{paddedNumber}`

Examples:
- `base1-004` - Base Set Charizard
- `jungle-010` - Jungle Scyther
- `fossil-003` - Fossil Ditto
- `base2-004` - Base Set 2 Charizard
- `rocket-004` - Dark Charizard
