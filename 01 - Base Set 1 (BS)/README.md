# Base Set (BS)

The original Pokemon Trading Card Game set that started it all. Released on January 9, 1999, Base Set introduced the world to the Pokemon TCG with 102 cards featuring iconic Pokemon like Charizard, Blastoise, and Venusaur.

## Notable Cards

- **Charizard** (#004) - The most iconic and valuable card from the set
- **Blastoise** (#002) - Key card in the dominant "Rain Dance" archetype
- **Alakazam** (#001) - Damage Swap Pokemon Power enabled unique strategies
- **Computer Search** (#071) - One of the most powerful Trainer cards ever printed
- **Professor Oak** (#088) - Essential draw power for every deck
